## 1️⃣ 双指针

### 场景

* 去重 / 移动

必刷：

* 移动零
* 删除有序数组重复项

---

## 2️⃣ 滑动窗口（🔥前端最重要）

### 本质

```js
for (right) {
  while (不满足条件) left++
}
```

必刷：

* 最长无重复子串（🔥必会）
* 最小覆盖子串

👉 这类题：
= **字符串处理 / 输入框 / 搜索优化 的抽象**

---

## 3️⃣ 二分查找

必刷：

* 二分模板（左右边界）
* 搜索旋转数组

👉 前端对应：

* 虚拟列表定位
* 大数据查找优化

# 🧠 Phase 3：算法思想（第4–6周）

这是决定你“能不能进大厂”的核心

---

## 1️⃣ DFS / 回溯（重点）

### 模型

```js
function backtrack(path, choices) {
  if (结束条件) return

  for (choice of choices) {
    做选择
    backtrack()
    撤销选择
  }
}
```

必刷：

* 全排列
* 子集
* 组合总和

---

## 2️⃣ BFS（树/图）

必刷：

* 二叉树层序遍历
* 最短路径（简单图）

---

## 3️⃣ 动态规划（🔥核心难点）

### 你只需要掌握 3 类：

1. 线性DP

   * 爬楼梯
   * 打家劫舍

2. 子序列

   * 最长递增子序列

3. 字符串DP

   * 最长回文子串

---

👉 DP本质你要理解：

```
状态 + 转移 + base case
```
